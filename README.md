# CIPHERCHAT

CipherChat is a full-stack real-time chat application built with React, Node.js, Express, and MongoDB. It features end-to-end messaging, friend requests, real-time status updates, and group chats.

## Features

- **Real-time Messaging**: Instant message delivery using Socket.io.
- **Authentication**: Secure user authentication with JWT.
- **Friend System**: Send, accept, and manage friend requests.
- **Group Chats**: Create and manage group conversations.
- **User Status**: Real-time online/offline/away status updates.
- **Modern UI**: Cyber-themed responsive interface.

## Project Structure

This repository contains both the frontend and backend of the application:

- `/frontend`: React application.
- `/backend`: Node.js/Express server with MongoDB integration.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Nithish0333/CIPHERCHAT.git
   ```

2. Install Backend dependencies:
   ```bash
   cd backend
   npm install
   ```

3. Install Frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Running the Application

1. Start the Backend:
   ```bash
   cd backend
   npm start
   ```

2. Start the Frontend:
   ```bash
   cd frontend
   npm start
   ```

## Branches

- `main`: Contains the full project (frontend and backend).
- `frontend`: Contains only the frontend application.
- `backend`: Contains only the backend application.
